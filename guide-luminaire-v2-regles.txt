  FORMATS DE LUMINAIRE (nouvel attribut pa_format)

  │ Format     │ Description              │ Contrainte               │
  ├────────────┼──────────────────────────┼──────────────────────────┤
  │ Horizontal │ Large, peu de hauteur    │ Aucune — OK partout      │
  │            │                          │                          │
  │ Boule      │ Forme ronde / sphérique  │ Aucune — OK partout      │

  ATTRIBUTS WOOCOMMERCE À CRÉER
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. pa_format (attribut produit)
     Valeurs : horizontal, boule, vertical
     Sur : toutes les suspensions (minimum)
     Rôle : exclure les verticaux si plafond bas sans table

  2. pa_type-ampoule (attribut produit)
     Valeurs : degagee, protegee
     Sur : tous les produits
     Rôle : filtrer selon la pièce / l'usage

  3. Variation "avec prise" (nouvelle variation)
     Sur : toutes les appliques
     Rôle : proposer des appliques même sans sortie murale

  LES 3 FINALITÉS — Quand orienter vers quoi ?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ACHAT DIRECT — quand :
    - Le client est convaincu par la recommandation
    - Ses besoins collent parfaitement à un modèle existant
    - Il n'a pas de demande particulière
    → Bouton vers la fiche produit

  CONTACT ROBIN — quand :
    - Le client hésite entre deux modèles
    - Questions techniques (installation, ampoule, entretien)
    - Situation un peu atypique mais pas besoin de sur-mesure
    → Formulaire pré-rempli avec le contexte de la conversation

  SUR-MESURE — quand :
    - Rien dans le catalogue ne correspond vraiment
    - Envie spécifique (dimension, forme, combinaison)
    - Le client dit "j'aimerais un truc entre les deux"
    → Brief structuré (forme, taille, essence, usage) envoyé à Robin

  Toujours présents :
    - "J'hésite, aidez-moi à choisir"
    - "Je voudrais quelque chose d'unique"

  Contextuels (selon les réponses) :
    - Si plusieurs produits → "Quelle différence entre ces modèles ?"
    - Si grande pièce      → "Comment disposer plusieurs luminaires ?"
    - Si format exclu      → "Et si j'adapte la hauteur de suspension ?"

  Orienté sur-mesure :
    - "Et si on créait un luminaire sur-mesure ?"

  TECHNIQUE — CLÉ API ET MODÈLES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Clé API :
    - Stockée dans wp-config.php : define('ANTHROPIC_API_KEY', 'sk-ant-...')
    - JAMAIS exposée côté client

  Modèles Claude à utiliser :
    - Recommandation initiale → Sonnet (~0.01€ / appel, plus réfléchi)
    - Conversation de suivi   → Haiku  (~0.001€ / message, rapide)

  Protection anti-abus :
    - Max 15 messages par session
    - Rate limiting par IP
    - Nonce WordPress
