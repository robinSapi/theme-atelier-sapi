Robin Conseiller : l'impact de chaque question

Comment chaque réponse du visiteur agit concrètement sur la sélection de luminaires.
Mis à jour le 11 juin 2026 (conforme au code actuel)

1.Le moteur en bref

Le visiteur répond à des questions, ces réponses deviennent des filtres appliqués au catalogue, on obtient une liste de produits, le code en choisit 4, et seulement à la fin l'IA rédige le texte.

1. QuestionsLe visiteur répond (quiz, modale ou texte libre)
→
2. Filtres PHPCatégorie, format, ampoule, taille, essence
→
3. Liste produitsLes modèles qui passent les filtres
→
4. Choix des 4pick_four : le code choisit
→
5. L'IA rédigeLe texte de conseil, pas la sélection
Le point clé à retenir : au premier tour, c'est le code (PHP) qui choisit les 4 produits affichés, pas l'IA. L'ordre est fixe : 1) la meilleure vente, 2) le dernier modèle ajouté, 3) la 2e meilleure vente, 4) un modèle au hasard parmi le reste. L'IA ne fait qu'écrire le texte d'accompagnement. Elle ne choisit des produits que dans deux cas : la conversation de raffinement (« refine ») et le champ de texte libre de la page Mes créations. Donc pour améliorer la pertinence des suggestions, le levier est dans les filtres et dans pick_four, pas dans les consignes données à l'IA.

Les trois traduisent les réponses dans les mêmes « slugs » (cuisine, plafond, moderne…) et passent par les mêmes fonctions de filtrage. Le reste de ce document décrit ces filtres communs.

3.L'impact de chaque question

Une fiche par question. Les pastilles colorées indiquent les leviers que la question actionne.

Question 1

Pour quelle pièce ?

Toujours posée, en premier.
Type d'ampoule Catégorie (cuisine) Format (escalier) Étape suivante Sur-mesure (escalier)

Surprise utile : la pièce ne choisit pas le type de luminaire (c'est la question « Où installer ? » qui le fait). La pièce sert surtout à régler le type d'ampoule.

RéponseEffet sur le filtrage
CuisineAmpoule dégagée ou semi-dégagée. Jamais de lampe à poser (retirée des catégories). En grande pièce, ce filtre ne s'applique plus : tous les types d'ampoule sont acceptés.
Bureau / AtelierAmpoule dégagée ou semi-dégagée. En grande pièce, ce filtre ne s'applique plus : tous les types d'ampoule sont acceptés.
Salon / Salle à mangerAmpoule entourée ou semi-dégagée.
ChambreAmpoule entourée ou semi-dégagée.
Chambre enfantTraitée comme une chambre (entourée ou semi-dégagée).
Entrée / CouloirAmpoule entourée ou semi-dégagée (aligné sur salon/chambre depuis peu). Autorise le format vertical si la hauteur est confortable ou haute.
Cage d'escalierAucun filtre ampoule. Bascule vers « Quel type d'escalier ? » au lieu de la taille classique. Autorise le format vertical, exclut l'horizontal. Déclenche le parcours sur-mesure dans la modale.
Question 2

Quelle taille fait ta pièce ?

Posée pour toutes les pièces sauf l'escalier (qui a sa propre question).
Taille de variation Format Étape suivante Sur-mesure (grande)
RéponseEffet sur le filtrage
PetitPré-sélectionne la plus petite taille disponible du produit. Combinée à une hauteur confortable ou haute, autorise le format vertical. Saute la question « Ce sera l'éclairage principal ? » (réservée aux grandes pièces) et va directement à « Où installer ? ».
Standard (moyenne)Pré-sélectionne la 2e taille du produit (la 1re s'il n'en a qu'une). Saute la question « Ce sera l'éclairage principal ? » (réservée aux grandes pièces).
GrandPré-sélectionne la 3e taille. Exclut les suspensions qui ont 2 tailles ou moins (filtre relâché si plus rien ne reste). Ouvre l'étape Éclairage. Déclenche la carte sur-mesure.
Je ne sais pasAucun filtre de taille : tous les produits passent, quelle que soit leur taille.

Ce sera l'éclairage principal ?

Posée seulement si la pièce est grande.
Catégorie (remplace le choix par sortie)
RéponseEffet sur le filtrage
Source principaleFiltrage normal : la catégorie est décidée par la question « Où installer ? ».
J'ai d'autres sources
(secondaire)		Bascule sur un pool réduit (lampadaires + lampes à poser + appliques), puis affiné par la sortie : plafond → suspensions en complément ; mur → appliques ; prise 230V ou inconnue → lampadaires + lampes + appliques. C'est cette question qui décide du type de luminaire affiché, pas la pièce.

Question 5

Quelle hauteur sous plafond ?

Posée seulement si la sortie est au plafond (et hors escalier).
Format Étape suivante Sur-mesure
RéponseEffet sur le filtrage
Standard (< 2,50 m)Ouvre l'étape « Au-dessus d'une table ? ». Le format vertical reste exclu par défaut.
Confortable (2,50 à 3 m)Saute l'étape Table. Autorise le format vertical si la pièce est une entrée ou une petite pièce. Déclenche la carte sur-mesure.
Haute (> 3 m)Autorise le format vertical dans tous les cas (toutes pièces, toutes tailles). En petite pièce, exclut aussi le format horizontal. Saute l'étape Table. Déclenche la carte sur-mesure.
Question 6

Au-dessus d'une table ou d'un îlot ?

Posée si la hauteur est standard (hors entrée et escalier).
Aucun filtre aujourd'hui
Attention : dans le code actuel, cette réponse ne change pas la sélection de produits. Elle n'est utilisée que dans le texte de contexte envoyé à l'IA pendant une conversation de raffinement. En plafond + hauteur standard, le format vertical est de toute façon déjà exclu, que la réponse soit « oui » ou « non ». C'est donc une question devenue cosmétique : à revoir, soit pour lui redonner un effet, soit pour la retirer.

5.Pièges et points à surveiller

La question « table » ne filtre rien, et la boule non plus

La réponse « table » ne sert qu'à colorer le texte de l'IA en raffinement : la sélection est identique que le visiteur réponde oui ou non. Le format boule, lui, n'est jamais filtré non plus, une boule passe dans tous les cas. Décision prise (juin 2026) : on garde la boule toujours autorisée, donc cette question reste sans effet sur la sélection pour l'instant.

L'option « multi-ampoules / grappe » est du code orphelin

Le questionnaire ne propose plus que « principale » et « secondaire » pour l'éclairage. Mais le code qui gère la grappe (carte sur-mesure dédiée, sélection d'un modèle par format) existe toujours et n'est plus atteignable. À nettoyer, ou à rebrancher si l'on veut réintroduire ce parcours.

La carte sur-mesure se déclenche différemment selon la porte d'entrée

Quiz : grande pièce, ou hauteur confortable / haute (ou grappe). Modale Conseiller : cage d'escalier, ou grande pièce combinée à une hauteur confortable / haute. Une grande pièce seule déclenche la carte dans le quiz mais pas dans la modale. À harmoniser si l'on veut un comportement unique.

Le filet de sécurité quand aucun produit ne passe

Si les filtres ne renvoient rien, ils sont relâchés en cascade : on enlève d'abord le filtre d'ampoule, puis le filtre de format, puis le filtre de taille. Chaque relâchement ajoute une note d'avertissement transmise à l'IA pour qu'elle signale honnêtement le compromis au visiteur.
