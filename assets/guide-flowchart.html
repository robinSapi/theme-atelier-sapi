2 — Taille
Quelle est la taille de votre pièce ?
Petite (< 10 m²)→ Sortie
Moyenne (10–20 m²)→ Sortie
Grande (> 20 m²)→ Éclairage (NEW)
Filtrage taille du luminaire :
  • Petite → plus petite taille disponible
  • Moyenne → 2e taille
  • Grande → 3e taille (exclut produits avec ≤ 2 tailles)
2b — Éclairage (si grande)
Pour un grand espace, plusieurs sources lumineuses font toute la différence.
Je cherche mon éclairage principal→ Sortie
Je cherche un éclairage secondaire→ Sortie
Je voudrais plusieurs ampoules→ Style (sur mesure)
Je verrai plus tard→ Sortie
Impact sur le filtrage :
  • Principal — filtre normal (suspensions, etc.)
  • Secondaire — lampadaires, lampes à poser, appliques
  • Plusieurs ampoules — sur mesure "en grappe", pas de filtrage produit
  • Plus tard — filtre normal, comme "principal"
3 — Sortie électrique
Où se trouve votre sortie électrique ?
Au plafond→ Hauteur
Au mur→ Style (skip 4-5)
Sur prise classique 230V→ Style (skip 4-5)
Je ne sais pas→ Style (skip 4-5)
Impact "Éclairage secondaire" :
  • Si secondaire → propose lampadaires, lampes à poser, appliques (pas suspensions)
  • La question "Sortie" est alors sautée (pas pertinent)
4 — Hauteur (si plafond)
Quelle est votre hauteur sous-plafond ?
Standard (< 2,50 m)→ Table*
Confortable (2,50–3 m)→ Style (skip 5)
Haute (> 3 m)→ Style (skip 5)
Filtrage format vertical :
  • Vertical EXCLU par défaut pour suspensions
  • SAUF : escalier (toute hauteur)
  • SAUF : couloir + hauteur grande ou confortable
  • SAUF : petite pièce + hauteur grande ou confortable
5 — Table (si standard)
Au-dessus d'une table / du lit / du bureau ?
Oui→ Style
Non→ Style
Question adaptée selon la pièce :
  • Cuisine — "Au-dessus d'une table ou d'un îlot ?"
  • Bureau — "Au-dessus du bureau ?"
  • Salon — "Au-dessus d'une table ?"
  • Chambre — "Au-dessus du lit ?"
* Entrée, couloir et escalier : question sautée
6 — Style
Quel est le style de votre intérieur ?
Moderne, neuf, tons clairs→ Résultat
Ancien, pierre, bois, tons chauds→ Résultat
Filtrage essence :
  • Moderne → variation Peuplier (clair)
  • Ancien → variation Okoumé (chaud)
Résultat
Sélection personnalisée
Texte IA signé "— Robin" (max 80 mots) Explique pourquoi cette sélection correspond Grille de 4 produits filtrés (1 ligne desktop) Chaque carte : image, nom, prix, essence + taille recommandées, CTA
Sélection des 4 produits (si > 4 filtrés) :
  • 1) Le plus vendu
  • 2) Le dernier ajouté au catalogue
  • 3) Le 2e plus vendu
  • 4) Aléatoire parmi les restants

Parcours types

Parcours complet — salon standard (7 étapes)
Salon → Grande → Éclairage principal → Plafond → Standard → "Au-dessus d'une table ?" Oui → Moderne → Résultat (4 produits)
Parcours grande pièce — éclairage secondaire (5 étapes)
Salon → Grande → Éclairage secondaire → sortie → hauteur → table → Moderne → Résultat (lampadaires, lampes, appliques)
Parcours grande pièce — multi-ampoules (4 étapes)
Chambre → Grande → Plusieurs ampoules → sortie → hauteur → table → Ancien → Résultat (sur mesure "en grappe")
Parcours court — cage d'escalier (3 étapes)
Cage d'escalier → taille → Mur → hauteur → table → Ancien → Résultat
Parcours moyen — plafond haut (5 étapes)
Chambre → Petite → Plafond → Haute → table → Moderne → Résultat
Parcours moyen — prise classique (4 étapes)
Cuisine → Moyenne → Sur prise 230V → hauteur → table → Ancien → Résultat
