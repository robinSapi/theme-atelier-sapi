↗ Sortie saute Éclairage

Filtrage taille luminaire :
  • Petite → plus petite taille disponible
  • Moyenne → 2e taille
  • Grande → 3e taille (exclut produits ≤ 2 tailles, fallback si vide)
2b
Éclairage Si grande pièce

Pour un grand espace, Robin recommande toujours de multiplier les sources lumineuses. Que pensez-vous ?

↓ Sortie
Éclairage principal Compléter existant (secondaire) Je verrai plus tard
↗ Style sur mesure — saute Sortie/Hauteur/Table
Multi-ampoules (grappe)
Impact éclairage :
  • Principal / Plus tard → filtrage normal
  • Secondaire → pool limité (lampadaires + lampes + appliques), affiné par sortie
  • Grappe → résultat sur mesure "en grappe" + CTA contacter Robin

Où se trouve votre sortie électrique ?

Visible quand :
  • Petite ou Moyenne (skip éclairage)
  • Principal, Secondaire ou Plus tard (après éclairage)
  • Escalier (skip taille + éclairage)
↓ Hauteur
Au plafond
↗ Style saute Hauteur + Table
Au mur Prise 230V Je ne sais pas
Filtrage catégories :
  • Plafond → suspensions
  • Mur → appliques
  • Prise 230V → lampadaires + lampes à poser + appliques
  • NSP → suspensions + lampadaires + lampes (pas d'appliques)
  • Secondaire + Plafond → suspensions (IA mentionne les autres en complément)
  • Secondaire + Mur → appliques seules
  • Secondaire + Prise 230V → lampadaires + lampes + appliques
  • Secondaire + NSP → lampadaires + lampes (pas d'appliques)
4
Hauteur Si sortie = plafond

Quelle est votre hauteur sous-plafond ?

↓ Table sauf escalier & entrée
Standard (< 2,50 m)
↗ Style saute Table
Confortable (2,50–3 m) Haute (> 3 m)
Format vertical (suspensions) :
  • Exclu par défaut
  • Autorisé si escalier (toute hauteur)
  • Autorisé si entrée/couloir + confortable ou haute
  • Autorisé si petite pièce + confortable ou haute
5
Table Standard + pièce ≠ escalier/entrée

Sera-t-il au-dessus d'une table ou d'un îlot ?

↓ Style
Oui Non
Question dynamique selon pièce :
  • Cuisine → "Au-dessus d'une table ou d'un îlot ?"
  • Bureau → "Au-dessus du bureau ?"
  • Salon → "Au-dessus d'une table ?"
  • Chambre → "Au-dessus du lit ?"
6
Style Toujours

Quel est le style de votre intérieur ?

↓ Résultat
Moderne, neuf, tons clairs Ancien, pierre, bois, tons chauds Pas de préférence
Filtrage essence :
  • Moderne → variation Peuplier (clair)
  • Ancien → variation Okoumé (chaud)
  • Neutre → pas de filtre essence
★
Résultat

Sélection personnalisée

Contenu du résultat :
  • Texte IA signé "— Robin" (max 80 mots, pas de markdown)
  • Grille de 4 produits filtrés (image, nom, prix, essence + taille reco.)
  • Sélection : 1) best seller, 2) dernier ajouté, 3) 2e best seller, 4) aléatoire
  • OU résultat sur mesure "en grappe" + CTA contacter Robin (si grappe)
Règles spéciales :
  • Cuisine → jamais de lampe à poser
  • Escalier → exclut format plat
  • Fallback : si 0 résultats → relâche filtre ampoule, puis format, puis taille
