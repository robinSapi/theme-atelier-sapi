## BENCHMARK SUMMARY

Le benchmark (bōlum, Loupiote, HAY, Moooi, Vitra, Bellroy, Aesop) révèle **8 patterns dominants** chez les e-commerces premium artisanaux :

1. **Galerie 50/50 généreuse** : Image principale ≥500px, vignettes ≥100px, zoomable
2. **Prix dynamique temps réel** : Le prix exact s'affiche instantanément selon la variation sélectionnée
3. **Rassurance sous CTA** : 3 icônes + textes courts (fabrication, livraison, retours) dans la zone de décision
4. **Pitch émotionnel** : 2-3 lignes de storytelling sous le titre, PAS après le formulaire
5. **Section Artisan** : Photo du créateur + 2 phrases sur l'histoire de ce modèle spécifique
6. **Lifestyle contrôlé** : Maximum 2-3 images (≤60vh), TOUTES montrant le produit
7. **Tableau specs scannable** : Dimensions, matériaux, ampoule — format grille, pas de prose
8. **Video/GIF du produit** : Montrer la lumière en mouvement (on vend de la LUMIÈRE)

**Conclusion clé** : Les sites qui convertissent placent le storytelling et la rassurance DANS la zone de décision (above the fold), pas après. Le lifestyle vient APRÈS la conversion, pas avant.

#### Patterns benchmark utilisés

| Pattern | Source | Application |
|---------|--------|-------------|
| Video Hero | **Moooi** | GIF/video du luminaire allumé en ouverture (70vh) |
| Pitch émotionnel | **Aesop** | 2 lignes de copywriting poétique dans la zone conversion |
| Rassurance sous CTA | **bōlum** | 3 icônes fabrication/livraison/retours sous le bouton |
| Section Artisan | **Ferm Living** | "Mot de Robin" avec photo atelier |
| Galerie scroll verticale | **HAY** | Photos empilées verticalement, vignettes en bas |
| Specs tableau | **HAY** | Grille 3×2 scannable |

#### Pourquoi ça convertit

1. **Hero video = impact émotionnel immédiat** → Le client voit la lumière en action AVANT de réfléchir au prix. Émotion > Raison.

2. **Zone conversion compacte avec rassurance intégrée** → Tout ce dont le client a besoin pour décider est visible SANS scroller : pitch, prix dynamique, variations avec labels, CTA, et rassurance. Réduction de la friction cognitive.

3. **Storytelling artisan = différenciation** → Le "Mot de Robin" répond à la question "Pourquoi acheter ici plutôt que chez IKEA ?". Le client paie pour l'histoire, pas juste pour un luminaire.

4. **Lifestyle APRÈS conversion** → Les images d'ambiance viennent après le formulaire, pas avant. Le client a déjà ajouté au panier quand il les voit = confirmation de son choix, pas distraction.

5. **CTA sticky mobile** → Sur mobile, le bouton d'achat reste toujours visible avec le prix exact. Taux de conversion mobile x2.

#### Patterns benchmark utilisés

| Pattern | Source | Application |
|---------|--------|-------------|
| Configurateur visuel | **Bellroy** | Image produit qui change selon la config sélectionnée |
| Boutons taille avec prix | **Vitra** | Chaque taille affiche son prix directement |
| Cards matériau avec description | **Ferm Living** | "Okoumé - Bois clair doré" au lieu d'un simple cercle |
| Livraison estimée | **HAY** | "Livraison estimée : 12 fév." (date calculée) |
| Double CTA | **Bellroy** | "Ajouter" + "Acheter maintenant" (skip cart) |
| Accordéon specs | **Aesop** | Infos détaillées sans surcharger la page |

#### Pourquoi ça convertit

1. **Configurateur interactif = engagement** → Le client "joue" avec les options. Chaque clic crée un investissement émotionnel. Plus il configure, plus il veut acheter (effet IKEA inversé).

2. **Prix visible sur chaque option** → Zéro surprise. Le client sait que 90cm = 160€ AVANT de cliquer. Transparence = confiance = conversion.

3. **Double CTA stratégique** → "Acheter maintenant" pour les décideurs rapides (skip le panier). "Ajouter au panier" pour les comparateurs. On capture les deux profils.

4. **Livraison estimée personnalisée** → "Chez vous le 12 février" est 10x plus convaincant que "Livraison 48h". Le client visualise le luminaire CHEZ LUI à une date précise.

5. **Accordéon = page épurée** → Les détails techniques sont là pour qui les veut, mais ne polluent pas la zone de décision. Mobile-friendly.

6. **Lifestyle "Vu chez vous"** → Framing psychologique : ce n'est pas "une photo de salon", c'est "VOTRE futur salon". Projection + conversion.
